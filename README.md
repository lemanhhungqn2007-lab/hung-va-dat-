# 🎨 README — CSS Styling for Seijaku Landing Page

## 🍵 Giới thiệu

File CSS của dự án **Seijaku** được xây dựng nhằm tạo nên giao diện mang phong cách trà đạo Nhật Bản với cảm giác:

* Tối giản

# ✨ Hiệu ứng CSS nổi bật

## Hover Animation

```css id="r7t4mi"
transform: translateY(-6px);
transition: 0.3s;
```

Giúp card và button chuyển động mượt khi rê chuột.

---

## Responsive Layout

```css id="pz7hwr"
@media(max-width: 900px)
```

Website tự động co giãn trên:

* Desktop
* Tablet
* Mobile

---

## Flexbox & Grid

Sử dụng:

* `display: flex`
* `display: grid`

để xây dựng layout hiện đại và dễ responsive.
